# Demochain | Algorand voting contract.

Lanzar el contrato en local:
1. Borrar `.venv` si existe.
2. Ejecutar `algokit project bootstrap all`
3. Ejecutar `añgokit project run build` dentro de la carpeta `demochain\voting-contract`

Una vez realizados los siguientes pasos ya tendreis el _Smart Contract_ compilado a lenguaje `TEAL`.
